# Preflight prompt pack — Absence detector (copy-paste)

Use in Cursor with Omega project open. Replace the bracketed section and send.

---

Run as OMEGA FORGE.

**ABSENCE DETECTOR**

**Domain or space (e.g. "clinical decision support", "my product category", "fundraising for pre-seed"):** [PASTE HERE]

Apply FORGE Absence Detector:

- What is conspicuously **missing** from this space?
- What does nobody build, ask, measure, or name — and why?
- The absence is often more informative than what is present. Most important innovations fill gaps that were invisible.

Output a single artifact:

1. **3–5 GAPS**  
   For each: what's missing (one line), why it might be invisible (one line), one way to test if it's real (one line).

2. **MOST UNDERRATED GAP**  
   Which gap, if filled, would have the highest leverage? One sentence.

3. **ONE QUESTION NOBODY ASKS**  
   A question that would meaningfully advance understanding in this space. Not rhetorical — answerable with effort.

Output: markdown, copy-paste ready. No filler.
