# Omega Ecosystem Apps Summary

## Core Applications

**omega-f**: Governance determination infrastructure assessing system governability using constitutional principles. Determines if systems are structurally governable without making policy recommendations.

**reflexive/reflexive-ui**: Institutional reasoning engine analyzing decisions through multiple analytical lenses (financial, legal, operational, strategic). Backend API (TypeScript/Fastify) with Next.js UI visualizing reasoning graphs, evidence, and traceability.

**omega-site**: Main Omega website/documentation built with Next.js and MDX for protocol documentation and system overview.

**OPLAS**: Post-LLM abstraction system building verifiable, executable abstractions that persist across tasks. Deterministic parse→canonicalize→hash→validate pipeline without LLMs in parse path.

**orientation-lab**: Non-authoritative thinking tool for navigating uncertainty and disagreement. Surfaces competing models, assumptions, and unknowns without forcing consensus or making recommendations.

**constraint-universe**: Formal verification engine using Z3 theorem prover. Classifies states as POSSIBLE/IMPOSSIBLE/INEVITABLE with mathematical proofs. Reconfigures worlds based on constraint changes.

**spine**: Contract system governing research→learning→evolution. Defines behavioral boundaries, integration contracts, and governance kernels for cross-project orchestration.

**trauma-informed-education-system**: Complete operating system for trauma-informed practice in schools. Self-contained framework with 8 modules, principles, and implementation guidance. Not therapy or clinical intervention.

**spine/spatial**: Human-in-the-loop spatial intelligence for exploring robotics/simulation/XR systems. Visualization and reasoning aid without autonomy or control.

**srfc**: Surgical Robotics Feasibility Compiler - deterministic rule-based feasibility assessment for surgical soft robotics using physics and constraints (no AI/ML).

**tsrfc**: Translational SRFC for procedures, failure surfaces, and evidence-adoption dynamics in spine/ENT/endoscopy contexts.

**vrfc**: VR/AR feasibility compiler evaluating evidence, regulatory, reimbursement, and adoption dimensions for medical procedures.

**omega-vision**: Swift/RealityKit spatial visualization tools for macOS/iOS exploring 3D scenes and spatial reasoning.

**omega-infra**: Docker-based infrastructure with backup/restore, health checks, and deployment automation for Omega services.
